# Disease Prediction System
A machine learning system that predicts Diabetes, Heart Disease, and Kidney Disease.

## Models Used
- Diabetes: XGBoost + SMOTE + Calibration
- Heart Disease: XGBoost + SMOTE + Calibration
- Kidney Disease: XGBoost + Calibration

## Datasets
- Diabetes: 100,000 patients
- Heart Disease: 246,022 patients
- Kidney Disease: 400 patients (UCI CKD Dataset)
